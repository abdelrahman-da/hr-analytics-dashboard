# HR Analytics Dashboard

## Tools Used
- Power BI
- DAX
- Data Modeling
- Power Query

## Project Overview

This HR Analytics Dashboard was developed to provide a comprehensive view of workforce performance, employee demographics, compensation analysis, and attrition trends.

The dashboard enables HR teams and decision-makers to monitor workforce KPIs, identify retention risks, analyze salary distribution, and explore employee demographics through interactive visualizations and drill-down capabilities.

## Business Objectives

- Monitor workforce performance and employee distribution.
- Analyze salary trends across departments.
- Track employee attrition and retention metrics.
- Understand workforce demographics and diversity.
- Support data-driven HR decision-making.

## Dashboard Pages

### 1. Summary View
Provides a high-level overview of key workforce metrics including:
- Total Employees
- Total Salaries
- Average Salary
- Average Age
- Average Bonus %
- Attrition Rate

### 2. Salary Analysis
Analyzes:
- Total Salaries by Department
- Average Salary by Department
- Top 10 Highest Paid Employees
- Bonus Distribution
- Department Compensation Comparison

### 3. Demographics Analysis
Provides insights into:
- Employee Age Groups
- Diversity Breakdown
- Workforce Distribution
- Hiring Trends Over Time

### 4. Attrition Analysis
Tracks:
- Attrition Rate by Department
- Attrition Rate by Age Group
- Exited Employees by Department
- Employee Retention Trends

### 5. Detailed Workforce View
Detailed workforce reporting including:
- Department Breakdown
- Gender Analysis
- Salary Distribution
- Employee-Level Insights
